# NETFLIX-Clone

**NETFLIX-Clone** is a **Netflix-style clone** built with **React** and **Firebase**. Users can **sign up, log in, and browse movies** fetched from a basic API. The app is fully **responsive** with interactive carousels, hero banners, and a clean UI.

---

## Features

- Firebase Authentication (Sign Up / Log In / Log Out)  
- Fetch movies and details from a basic API  
- Responsive design for desktop and mobile  
- Interactive movie carousels and hero banners  
- Clean and modern UI/UX similar to Netflix  

---

## Tech Stack

- **Frontend:** React, CSS  
- **Backend/Auth:** Firebase Authentication  
- **Database:** Firebase Firestore (optional for user info)  
- **API:** TMDb or any basic movie API
